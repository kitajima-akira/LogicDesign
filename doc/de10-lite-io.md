# FPGAボードDE10-Lite上の主な要素の利用

LogicDesignの既定の設定では、モジュールのポート名を以下のとおり決まった名前で指定することで、FPGAボード上の各要素との対応付けが行えます。
なお、ポート名が\_nで終わるポートは負論理となっていますので、記述中で正論理に直してから使用することを勧めます。

| ポート名 | 方向 | ビット幅 | 説明 | マニュアル中の名前 |
| ----- | :---: | :---: | ----- | ----- |
| `clock` | 入力 | 1 | クロック入力 |  |
| `reset_an` | 入力 | 1 | リセット入力 (負論理) |  |
| `KEY0_n, KEY1_n` | 入力 | 各1 | タクトスイッチ (負論理) | Button x2 |
| `SW` | 入力 | 10 | スライドスイッチ (10個分) | Switch x10 |
| `LEDR` | 出力 | 10 | LED (10個分) | LED x10 |
| `HEX0_n, HEX1_n, HEX2_n, HEX3_n, HEX4_n, HEX5_n` | 出力 | 各7 | 7セグメントLED (負論理) | 7-Segment Display x6 |
| `DP_n` | 出力 | 6 | 7セグメントLEDのドット部分 (負論理) |  |
